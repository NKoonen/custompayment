# Custom Payment module for PrestaShop

A lightweight and flexible PrestaShop module that allows you to create your own fully customizable payment option.  
Perfect for manual payments, offline methods, invoicing workflows, or any scenario where you need a custom checkout option without writing additional code.

## Features

- Create a custom payment method with your own:
    - Title
    - Description shown at checkout
    - Email template selection for order confirmation
    - Order status assigned after validation
- No core modifications
- Straightforward configuration inside the Back Office
- Compatible with PrestaShop **1.7**, **8** and **9.x**
- Supports multilingual content


## Usage

This module is primarily used by shops that allow existing customers to place orders on account, meaning payment is not made immediately but settled later by invoice typically monthly or at other agreed intervals, often for B2B clients with custom payment arrangements.

Once configured:

- The payment method will appear during checkout.
- The module applies your chosen order status after order confirmation.
- Customers receive your custom email template.
- No additional hooks or modifications required.
